# Bank Management System (BMS)

Bank Management System (BMS) is a Python project for basic banking operations such as creating accounts, depositing money, withdrawing money with balance checks, viewing balances, listing accounts, closing accounts, and modifying account details. Account data is stored locally in a file (`accounts.data`) using `pickle`.

Install and run:

```bash
git clone https://github.com/HarshShinde0/BMS.git
cd BMS
python bank.py
```

To run the Streamlit version:

```bash
pip install streamlit
streamlit run bms.py
```

How to use:

Run the program, and you will see the following options:

```
1. NEW ACCOUNT
2. DEPOSIT AMOUNT
3. WITHDRAW AMOUNT
4. BALANCE ENQUIRY
5. ALL ACCOUNT HOLDER LIST
6. CLOSE AN ACCOUNT
7. MODIFY AN ACCOUNT
8. EXIT
```

Enter the option number you want and follow the prompts.
