# Bank Management System (BMS)

Bank Management System (BMS) is a Python command-line project that helps manage basic banking operations. It supports creating, modifying, and deleting accounts, as well as depositing money, withdrawing money with balance checks, and viewing account details. Data is stored locally using file handling with `pickle`.

Install and run:

```bash
git clone https://github.com/HarshShinde0/BMS.git
cd BMS
pip install -r requirements.txt
python bank_management.py
```

How to use:

Run the program, and you will see the following options:

```
1. NEW ACCOUNT
2. DEPOSIT AMOUNT
3. WITHDRAW AMOUNT
4. BALANCE ENQUIRY
5. ALL ACCOUNT HOLDER LIST
6. CLOSE AN ACCOUNT
7. MODIFY AN ACCOUNT
8. EXIT
```

Enter the option number you want and follow the prompts.

Testing:

```bash
python -m unittest discover tests
```
